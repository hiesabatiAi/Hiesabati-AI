# Hiesabati AI Corporate Platform

Welcome to the **Hiesabati AI** web platform. This repository contains the front-end codebase for the official Hiesabati enterprise portal—a sophisticated, next-generation AI CFO and autonomous compliance engine designed for global enterprises.

## Live Demo : https://hiesabatiai.github.io/Hiesabati-AI/

## 🚀 Overview

Hiesabati AI integrates advanced machine learning models with robust double-entry financial ledgers to deliver complete operational visibility for international businesses. The platform handles:
- **Autonomous Compliance Engines:** Running continuous ledger audits and tax risk tracking (VAT, corporate tax, regional customs).
- **Aria — Corporate AI CFO:** An intelligent simulator providing real-time financial insights, variance reconciliation, and cash flow projections.
- **Global Data Streams:** A dynamic Sovereign Audit Risk Dashboard mapping live ledger synchronizations across regional entities.

## 🎨 Design & Architecture

The site is built with a premium, corporate fintech aesthetic featuring:
- **Glassmorphism UI:** Sophisticated glowing cards, deep navy backgrounds, and sleek borders.
- **Dynamic Data Visualizations:** High-performance HTML5 Canvas animations (e.g., interactive 3D global particle networks, AI voice waveforms).
- **Responsive Grid Systems:** Seamless transition from ultra-wide enterprise monitors down to mobile devices.
- **Dark/Light Mode:** Full structural theme support tailored for institutional environments.

## ⚡ Performance Optimizations

The codebase is engineered for maximum performance:
- **Smart Viewport Observing (IntersectionObserver):** Heavy background animations (like the canvas globe and AI waveforms) automatically pause when scrolled out of view to minimize CPU overhead.
- **Mathematical Optimization:** Particle animations use squared-distance calculations to ensure a buttery-smooth 60fps experience.
- **Debounced Listeners:** Scroll events and layout shifts are properly throttled using `requestAnimationFrame`.

## 🛠️ How to Run

Because this is a static front-end application built with native Web Technologies (Vanilla HTML, CSS, JavaScript), there is no heavy build process required.

1. Clone or download this repository.
2. Open `index.html` in any modern web browser (Chrome, Edge, Safari, Firefox).
3. (Optional) Run a local development server for a better experience:
   ```bash
   npx serve .
   # or
   python -m http.server 8000
   ```

## 📂 Project Structure

- `index.html`: The core architectural structure and layout of the platform.
- `style.css`: The comprehensive design system containing variables, themes, animations, and responsive breakpoints.
- `app.js`: The central logic controller handling interactive components, canvas animations, mock data feeds, and theme switching.
- `assets/`: Contains all premium SVG icons, brand assets, and generated AI illustrations (e.g., `audit_map.jpg`).
